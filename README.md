# Jan Report Marketing Workspace

This repository contains the web dashboard reports generated from the marketing data.

## Available reports

- [January report](web_report/index.html) — Executive dashboard for January 2026 (main and revenue pages)
- [February report](web_report_feb/index.html) — Executive dashboard for full February 2026 dataset (main and revenue pages)
- [February date-range report](web_report_feb/index-range.html) — Interactive version allowing selection of any date range from the February dataset

You can open the HTML files directly in your browser. For example, in Windows Explorer right-click the file and choose "Open with > Browser" or copy the file path and paste into your browser:

```
file:///C:/Users/PC/Desktop/Jan%20Reoprt%20Marketing/web_report/index.html
```

Feel free to update or regenerate the reports by running the Python scripts.
